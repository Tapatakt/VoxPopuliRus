# Vox Populi Russian Translation

Русский перевод мода [Vox Populi](https://forums.civfanatics.com/forums/community-patch-project.497/) (Community Patch Project) для Civilization V.

## Описание

Этот мод добавляет полный русский перевод для Vox Populi — крупнейшего мода для Civilization V, который включает:
- Community Patch (основные исправления и улучшения игры)
- Vox Populi (новые механики, баланс, ИИ)
- EUI Compatibility (улучшенный интерфейс)
- Squads for VP

Это нейросетевой перевод (использовалась Kimi Code 2.5) с лёгкой вычиткой. Наверняка в нём много косяков и неточностей. Я буду исправлять их по мере обнаружения. Вы можете помочь этому процессу, написав мне или создав Pull Request.

## Требования

- Civilization V (версия 1.0.3.279 или новее)
- [Vox Populi 5.1.4](https://forums.civfanatics.com/threads/vox-populi-5-1-4-release.689390/) или совместимая версия

## Установка

### Способ 1: Через ModBuddy (для разработчиков)

1. Склонируйте репозиторий
2. Откройте `VoxPopuliRus.civ5sln` в ModBuddy
3. Соберите проект (Build → Build Solution)

### Способ 2: Через Steam (для всех остальных)

[Вот же он!](https://steamcommunity.com/sharedfiles/filedetails/?id=3684232823)
